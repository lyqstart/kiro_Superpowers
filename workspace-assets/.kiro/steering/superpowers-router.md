# Superpowers Router

根据用户自然语言自动路由：

- 新增/增加/开发/实现/支持 → Feature Spec：requirements → design → tasks → TDD → verification。
- 修复/bug/报错/异常/失败/不对 → Bugfix：复现 → 根因 → 失败测试或替代验证 → 最小修复 → 回归验证。
- 继续/接着/下一个任务 → Task execution：定位当前 spec 和未完成 task → 读取 requirements/design/tasks → pre-task gate → 执行 → review → verify。
- 审查/review/检查代码 → Review：spec review → code review → test review。
- 是否完成/验收/能交付吗 → Verification：diff → test/build/lint → spec compliance → code quality → COMPLETE/NOT COMPLETE/BLOCKED。

如果意图不清，只问一个最小问题，优先给编号选项。不要要求用户写长提示词。
