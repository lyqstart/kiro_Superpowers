# 卸载 Kiro Superpowers Discipline v2.0.0

## 1. 删除项目级文件

在项目根目录删除：

```text
.kiro/steering/superpowers-*.md
.kiro/hooks/*sp-*.kiro.hook
.kiro/agents/sp-*.md
.kiro/scripts/sp-*.sh
.kiro/scripts/sp-*.ps1
```

Windows PowerShell：

```powershell
Remove-Item ".kiro\steering\superpowers-*.md" -Force
Remove-Item ".kiro\hooks\*sp-*.kiro.hook" -Force
Remove-Item ".kirogents\sp-*.md" -Force
Remove-Item ".kiro\scripts\sp-*.sh" -Force
Remove-Item ".kiro\scripts\sp-*.ps1" -Force
```

macOS/Linux：

```bash
rm -f .kiro/steering/superpowers-*.md
rm -f .kiro/hooks/*sp-*.kiro.hook
rm -f .kiro/agents/sp-*.md
rm -f .kiro/scripts/sp-*.sh
rm -f .kiro/scripts/sp-*.ps1
```

## 2. 卸载 Power

在 Kiro Powers 面板中移除：

```text
Kiro Superpowers Discipline / Superpowers Discipline for Kiro
```

## 3. 可选：删除解压目录

删除本安装包解压出来的目录即可。

## 注意

卸载不会删除你的业务代码、Kiro specs、git 分支或 worktree。worktree 清理必须由用户明确确认后单独执行。

## v2.0 Additional Cleanup

If installed, also remove:

```text
.kiro/superpowers-skills/
.kiro/superpowers-templates/
.kiro/superpowers-runtime/
```

`superpowers-runtime/` contains ledgers generated by this discipline. Delete it only if you no longer need those audit records.
