# Notice

This package adapts workflow ideas from the public Superpowers project by Jesse Vincent / obra.

It is not the original Superpowers plugin and does not install Superpowers into Kiro.
It translates selected execution disciplines into Kiro Power / Steering / Hooks / Custom Subagents.

Relevant upstream project:
- https://github.com/obra/superpowers

Superpowers is MIT licensed at the time this package was created.
This package contains newly written Kiro-focused guidance and does not attempt to mirror the original plugin implementation.
